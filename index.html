How the tracker works

The ASML Request Tracker provides an interactive view of service-request activity and operational performance. The Overview page presents total request volume, SLA compliance, average resolution time, active line-down requests, current request status and request-type distribution. The Operations page supports further investigation by showing request trends over time, resolution time by system model, workload by assigned engineer and individual request details. Use the Site Location, Request Type and Status controls to filter the report. Select a chart bar, segment or data point to cross-filter the other visuals, hover over items for additional information and use the report page navigation to move between the executive overview and operational detail. All displayed metrics recalculate automatically as filters are applied.

Demo data only: No production, confidential or client information is displayed.
